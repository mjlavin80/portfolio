Synthesis 1 Synthesis 2 Team Project Revised Project

1

The dataset I spent the most amount of time manipulating was for Synthesis 2. Using the Centers for Disease control and prevention was the biggest data set I have ever seen or worked with. Every single county in the United States had a data value every year for fifteen years. It was ‘UGE. I had to spend a bit of time restricting it so it was vertical data instead of the way it was given which was horizontal. Which meant I had to really create a workflow to incorporate all that data into a single column and to ensure it matched up with the corresponding year and county. After I processed that data I had to comb through it and delete the data values with totals for the state. I could not just have the program delete names of states because there are counties named after states. So I had to manually sift through all that data and delete the total state numbers because I did not need that data and I knew it would cause issues later when it came to the visualization aspect of the data. After deleting the state totals I had to match the counties with their respective unique code that represents them into a choropleth. This aspect was not entirely difficult. The most tedious aspect was going through and double checking the county and making sure it had the correct code.


I do not think my manipulation of this data really changed the audience's understanding of the data. I feel I kept the most important aspects, and still kept it mostly preserved, though it is easily read by a computer it is not easily read by a human. Even if you were to upload the CSV into an excel file the vertical data does not allow one to really understand it, unless they change it back to the horizontal format. I do appreciate the CDC representing the data in a horizontal format, even though it created more work for me, but it enabled me to quickly look at the data and create a general understanding of the dataset I am working with.


I did find data for Allegheny County for Lyme Disease but it was incomplete and I was not forthcoming with my trust into the data. The CDC data file, to me, was the best I was going to find and I still maintain the position. It was from a reputable source, had more information on how it achieved data collection and was very detailed in terms of the data listing.

2

For the team assignment, I worked with Amanda to produce the Texas execution timeline. Using an area line graph I felt would provide the maximum impact instead of a shiny line curving up and down. Using an area line graph also produced a grey ‘wall’ which helped the audience relate that to the grey walls in a prison setting. If I needed to reproduce this graph in a different way I would choose a bar graph, this would help with that ‘wall’ effect to the audience. This might be a way to show more ‘accurate’ numbers of executions per year. This though would lose the time aspect of the executions. These executions did not happen all in the same month or day, thus with the rigid lines of a bar graph, it would have created that effect.

3

In my Synthesis 2 project, I manipulated the color scheme so it was stacked. It varied from a light tan color to brown from 0-30 cases per year. I manipulated the color set to be more constrictive for the counties that were within the thirty cases. The counties that were hit hardest with Lyme disease easily surpasses one hundred and some reached into the five hundreds. When I first tackled this dataset I found code that read the range of data values and assigned colors accordingly. After using that color scheme it really did not give a story of the spreading that Lyme Disease has had in the past fifteen years. So from there, I started to narrow the color scheme and I found that eight colors helped display to the audience the spread across the northeast of Lyme Disease.


If I had to present my findings to an expert or science paper I would not choose a choropleth, for I feel there is some ambiguity in how it represents this dataset. Instead, I would choose dots to represent my data. Like I state bellow I think this helps represent the data more effectively and shows a complete picture of the data.

4

In terms of comparing multiple elements in my line graph for the Synthesis 1 project I graphed ‘Former Smoker’, ‘Smoking Tobacco Use’, and ‘Smoke Everyday’. Displaying the line graph horizontally helps the audience easily see there is a positive correlation between people who smoked every day and smoking tobacco use. One of the biggest issues with this data set that I found particularly troublesome was the aspect that in 2011 the Centers For Disease Control and Prevention changed their method of surveying. This caused an unnatural bump and dip in the graph. For my revision of a prior project, I did choose this project and revised the graph to have a hover effect and to display it with more of a curved line display and thin circles marking each year. I chose to do this because I wanted to still keep it relatively the same but challenge myself to change it to make it almost unrecognizable to showcase that the same data and essentially the same graph can be changed and become unique and different from its predecessor. If I was to make another revision I would create two line graphs, one 2005-2010 and then 2011-2015. This would help the data become more accurately represented and I would more easily be able to signify the surveys method change.

I do think my most proud aspect was Synthesis 2 with building the choropleth. Granted as I stated above I was not able to implement it into my website, but I still was able to build it and reflect my data. The choropleth is the proudest moment for I had never made anything like it before and to me, it was the most visually stunning.
